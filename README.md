# Учи.ру (uchi.ru) filter

## Как установить?
1. Заходим в настройки uBlock Origin 
2. Выбираем раздел "Список фильтров"
3. Листаем вниз и видим надпись "Импортировать"
4. Нажимаем на надпись, вставляем вот эту ссылку: <code>https://github.com/Je0121/uchi.ru_filter/raw/refs/heads/main/filter</code>
5. Нажимаем кнопку "Применить изменения".
6. Готово!
## Зачем этот фильтр нужен?
Стандартные фильтры uBlock Origin не блокируют рекламу на учи.ру, и мой фильтр это исправляет
